HTML Elements

When I first started Learning HTML, I couldn't differentiate between the words HTML elements and HTML tags so I sought for their meanings on Google. It turned out that they actually mean different things and at the same time I understood why people use them interchangeably.

An HTML element consist of its opening tag, atrributes, closing tag and also content to render to the browser. On the other hand an HTML tag(either opening or closing) is used to mark the start or end of an element.


<a href="https://www.google.com">
Go to google
</a>

In the illustration provided above we have a link element consisting of an open tag(<a>). Inside the open tag we have the href attribute that takes us to google upon clicking the content displayed by the browser, which is (Go to google) and then a close tag(</a>).

There are over 100 HTML tags and elements. Last I checked 111 HTML elements and tags. That's quite alot. The good new is we don't need all of them to be able to build a website.


Personally I don't believe you would use 100% of the tags in building any website of your choice.

In this section of the course we'll be treating a few of these tags, obviously the essentials you need to create a webpage. You can source for others on the internet or visit w3schools to learn more. There are other website you can learn from also.





<!DOCTYPE html>
<html>
<head>
<title>
My first Webpage
</title>
</head>
<body>
<h1>
Hello World!
</h1>
</body>
</html>

DOCTYPE html
The doctype tag is use to inform the browser that this document is an HTML document. It can be written in lowercase. Take note of the position of the exclamation mark "!".
html
The html tag is the tag that houses all other html elements. It follows immediately after the doctype declaration tag.
head
The head tag houses the title tag and meta data of the document. It follows after the html opening tag.
title
The title tag assigns a title to your document. It lives in the head tag.
body
The body tag houses every single visible content on your webpage. It follows after the closing tag of the head tag. Every other html tag goes in here.

Those were the essentials. Your webpage wouldn't display properly without them in place. Now lets see some of the body tags and their respective grouping.


Body tags can either be Block tags or Inline tags. A block level tag has a default horizontal break(empty line) after it, while an Inline tag doesn't exhibit this property.

Exmaples of Block Level tags include;

  • Paragraphs
  • Divs
  • Headings
  • Blockquote

Examples of Inline tags include;

  • Span
  • Text-formatting tags like emphasis, bold e.t.c.

There are quite alot of body tags. others include links, line-breaks, horizontal-rule, nav, header, main, footer, image, table, list, script, form, buttons, audio, video, iframes e.t.c.

Visit w3schools to learn more.
