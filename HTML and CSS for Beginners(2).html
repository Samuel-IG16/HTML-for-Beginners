HTML Elements and Attributes

When I first started Learning HTML, I couldn't differentiate between the words HTML elements and HTML tags so I sought for their meanings on Google. It turned out that they actually mean different things and at the same time I understood why people use them interchangeably.

An HTML element consist of its opening tag, atrributes, closing tag and also content to render to the browser. On the other hand an HTML tag(either opening or closing) is used to mark the start or end of an element.


<a href="https://www.google.com">
Go to google
</a>

I don't believe you would use 100% of the tags in building any website of your choice.

In the illustration provided above we have a link element consisting of an open tag <a>. Inside the open tag we have the href attribute that takes us to google upon clicking the content displayed by the browser, which is (Go to google) and then a close tag </a>.

There are over 100 HTML tags and elements. Last I checked 111 HTML elements and tags. That's quite alot. The good new is we don't need all of them to be able to build a website.

In this section of the course we'll be treating alot of these tags. We'll start with the essentials you need to create a webpage.


<!DOCTYPE html>
<html>
<head>
<title>
My first Webpage
</title>
</head>
<body>
<h1>
Hello World!
</h1>
</body>
</html>

The above written is referred to as the skeleton code of HTML. Now let's walk through this and explain what's actually going on.


DOCTYPE HTML
The doctype html tag is use to inform the browser that this document is an HTML document. It basically just tells the browser Hey! you are looking at an HTML file. It can be written in lowercase or all-caps. Take note of the position of the exclamation mark "!".
HTML
The html tag is the tag that houses all other html elements. It follows immediately after the doctype html declaration tag.
HEAD
The head tag houses the title tag (displays in the tab of your browser when you open your .html file), link tag to either your css or JavaScript files and meta tag that contains the meta data of the document. It follows after the html opening tag.
TITLE
The title tag assigns a title to your document. It lives in the head tag.
BODY
The body tag houses every single visible content on your webpage. It follows after the closing tag of the head tag. Every other html tag goes in here.

Those were the essentials. Your webpage wouldn't display properly without them in place. Now lets see some of the body tags and their respective grouping.

Paragraph Tag
The Paragraph tag p, is use to write a paragraph text. Its a Block-level element.
<p>
This is a paragraph

Link Tag
The Link tag is used to link an html document with a css document. The link tag is an empty element (it has no closing tag).
<link rel="stylesheet"> href="/css/master.css">

Line-break and Horizontal Rule
The Line-break br, tag is used to break into a new line while the Horizontal-rule hr tag is used to draw an horizontal line. They are both empty elements.
<!--This is a line-break-->
<br>
<!--This is an horizontal-rule-->
<hr>
