What is HTML?

Have you ever wondered how webpages are built?. Well the whole essence of this programming language (HTML) is to build websites(Web Development) using codes. HTML stand for Hyper Text Mark-up Language. Unlike Javascript and Python which are scripting languages, HTML is referred to as a '...Mark-up...' language because it identifies content with the use of tags.

 <p>
This is a paragraph.
</p>

The blue angle brackets are called tags. Most HTML Elements are composed of Open and Closed tags.

HTML is an easy programming language.

Create, Edit and Save

Creating an HTML document is very easy.

  1. Open the Text-editor of your choice.
  2. Create a new file.
  3. Save the file with .html extension instead of .txt

Editing an HTML document requires you to write codes in the file you just created. Let's write a simple message "Hello World" to the browser.

<!DOCTYPE html>
<html>
<head>
<title>
My first Webpage
</title>
</head>
<body>
<h1>
Hello World!
</h1>
</body>
</html>


Every HTML element in the example has an opening and closing tag except from the DOC-TYPE tag

Ensure to save the file before exiting the Text-editor. Open the file in your browser and the message "Hello World!" will be displayed as a header 1 text.

In this section of the course we were only interested in how to Create, Edit and Save HTML documents. Don't worry yourself too much about what the tags represent. In later sections I'll talk about HTML Elements in details.
