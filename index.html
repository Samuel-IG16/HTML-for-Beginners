How The Web Works

What happens when you are at home and you open up your browser and visit a website?

Let's break down the basic steps!

  • You start of by typing a URL into your browser e.g. www.google.com.
  • Then your computer sends this request as a packet, which includes the IP address of the website you want.
  • The IP address allows servers later on to help identify what website you are looking for.
  • It sends this request through wires, or a satellite which eventually links to wires using your internet service provider. So if you are at home your request is either sent through copper wires or optic fibre wires to the server you are about to reach or if you are on your cell phone, it links to satellite which links back down to earth which then ends up going to some sort of physical wire. On a basic level the internet is just wires connecting computers with some sort protocol.
  • So your ISP(Internet Service Provider) will then re-route the request through appropriate server location, using the IP address as the guide.
  • Once your request reaches the server it can send back the website you were asking for.
  • However a full website with content is too big to send as a single packet of data.
  • To solve this the server sends back the website split up into many packets.
  • The packets come with instructions on how to get back to you and reassemble once they reach you.
  • The packets don't actually care how they get to you just the final location so these packets may take different paths to get back to your computer or cell phone, what they really care about is the fastest way to get to your destination and your home IP address.
  • Once the packets reaches you they reassemble to show the page.
  • All of this moves at close to the speed of light, so it happens very fast.

What is HTML?

Have you ever wondered how webpages are built?. Well the whole essence of this programming language (HTML) is to build websites(Web Development) using codes. HTML stand for Hyper Text Mark-up Language. Unlike Javascript and Python which are scripting languages, HTML is referred to as a '...Mark-up...' language because it identifies content with the use of tags.

 <p>
This is a paragraph.
</p>

The blue angle brackets are called tags. Most HTML Elements are composed of Open and Closed tags.

HTML is an easy to learn programming language.

Create, Edit and Save

Creating an HTML document is very easy.

  1. Open the Text-editor of your choice.
  2. Create a new file.
  3. Save the file with .html extension instead of .txt

Editing an HTML document requires you to write codes in the file you just created. Let's write a simple message "Hello World" to the browser.

<!DOCTYPE html>
<html>
<head>
<title>
My first Webpage
</title>
</head>
<body>
<h1>
Hello World!
</h1>
</body>
</html>


Every HTML element in the example has an opening and closing tag except from the DOC-TYPE tag

Ensure to save the file before exiting the Text-editor. Open the file in your browser and the message "Hello World!" will be displayed as a header 1 text.

In this section of the course we were only interested in how to Create, Edit and Save HTML documents. Don't worry yourself too much about what the tags represent. In later sections I'll talk about HTML Elements in details.
